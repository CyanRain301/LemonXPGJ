# 🍋 柠檬的选片工具v1.0

轻量级照片筛选与管理助手，通过直观操作快速批量整理图片，支持选片表导入导出，实现工作进度保存与恢复。

---

## 🚀 快速开始

双击 `Start.bat`，等待后端服务启动后，浏览器将自动打开工具界面。

---

## 📖 使用指南

### 界面布局

从上到下依次为：

- **第一行**：导入图片文件路径输入框 + 导入图片按钮
- **第二行**：导出图片文件路径输入框 + 导出已选/未选图片按钮
- **第三行**：导入选片表 / 导出选片表按钮
- **展示区域**：图片缩略图网格

> 所有路径需填写完整本地绝对路径，如 `C:\Users\用户名\Pictures`

---

### 导入图片

1. 在输入框中填入文件夹路径
2. 点击「导入图片」
3. 工具将扫描该路径及其子目录下所有图像文件，自动转换并暂存 RAW 格式文件
4. 自动跳过不同子目录下的同名文件，仅保留首次出现的图片
5. 导入成功后，图片以缩略图网格形式展示

> 若发现同名文件，页面会弹窗提示，控制台将记录详细冲突列表

---

### 浏览与放大

- 鼠标滚轮上下滚动浏览缩略图
- 鼠标长按图片 → 临时放大查看细节 → 松开后自动恢复
- 放大操作不影响图片的保留/未选标记

---

### 图片状态标记

- 每张图片默认处于「保留」状态
- **右键操作**：
  - 右键点击图片 → 翻转状态（保留 ↔ 未选）
  - 长按右键并移动 → 批量翻转后续图片为同一状态

---

### 导出图片

1. 在「导出图片文件路径」输入框中填写目标文件夹
2. 点击「导出已选图片」或「导出未选图片」
3. 工具自动检测同名文件 → 跳过并提示冲突数量 → 其余正常导出
4. 可多次导出至不同位置，不影响原始图片

---

### 导出选片表

点击「导出选片表」→ 浏览器自动下载 `选片表_时间戳.txt`

文件格式：
```
[LEMON's PHOTO SELECT TOOL]
图片名1     save
图片名2     unsave
...
[END]
```

> 可直接用记事本编辑，便于版本管理与分享

---

### 导入选片表

1. 确保当前已导入相同一批图片
2. 点击「导入选片表」
3. 选择 `*.txt` 选片表文件
4. 工具自动解析并更新图片状态和图片更名行为
5. 展示区域立即刷新，反映新的标记结果和图片名

> 支持匹配成功数、未匹配文件等反馈信息

---

### 图片更名

- **鼠标左键**或**鼠标中键**点击图片名称 → 进入编辑状态
- 按 `Enter` 保存，按 `Esc` 取消
- 自动修正后缀名，导出时按新名称导出

---

## 📁 文件说明

| 文件/目录      | 说明                 |
| -------------- | -------------------- |
| `Start.bat`    | 启动脚本（双击运行） |
| `pics/`        | 图片缓存目录         |
| `选片表_*.txt` | 导出的选片进度文件   |

---

## 📌 注意事项

- 支持的图片格式：`.jpg`、`.jpeg`、`.png`、`.arw`、`.cr3` 等
- RAW 格式导入时自动转换为 JPG 预览，导出时还原为原始 RAW
- 同名文件将自动跳过
